[CSS] SCSS 사용법

2023-09-02

나는 프론트 개발할 때 Bootstrap은 절대 안쓰는 주의다.

Bootstrap 자체가 무거워서 성능에 영향이 있다는 이유도 있지만,

다른 디자인의 스타일시트를 갖다쓰는 것보다 내가 직접 만들어 쓰는 경우가 많아서 그렇다.

그래서 이제까지 CSS를 엄청나게 연마해왔는데...

이게 CSS 코드를 직접 짜다보면 선택자가 반복되는 게 너무 거슬렸었다.

찾아보니 SCSS라는게 있었고, 내가 불편했던 점들을 해소시켜준 거같다.

그 SCSS의 사용법을 적어보려 한다.

일단 SCSS는 브라우저에서 바로 사용할 수 있는 CSS로 컴파일해주는

CSS 전처리기 중 하나이다.

CSS 전처리기로는 SASS, Less, Stylus 등이 있다.

그래서 설치 모듈이 필요하다.

npm i -D sass node-sass sass-loader

SCSS 문법을 설명하기 위해 먼저 CSS로 된 예제를 보자.

#loading-wrap {
    position: absolute;
    top: 0;
    left: 0;
    display: table;
    background-color: rgba(255, 255, 255, 0.69);
    width: 100%;
    height: 100%;
    border-radius: 10px;
    z-index: 999999;
}

#loading-wrap > div {
    display: table-cell;
    vertical-align: middle;
}

#loading-wrap > div #loading {
    height: 70px !important;
    width: 70px !important;
    border: 6px solid #d2d2d2;
    border-right-color: #8d8d8d;
    border-top-color: #8d8d8d;
    border-radius: 100%;
    margin: auto;
    animation: spin 800ms infinite linear;
}

@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(359deg); }
}

위 스타일시트 코드는 실제로 회사 프로젝트로 진행했던 코드의 일부이다.

언뜻 보기엔 문제가 없어보인다. (당연히 돌아가는 데는 문제 없다)

근데 보다 보면 중간 중간에 살짝씩 반복되는 부분들이 보인다.

이런 부분들이 100라인이 넘어가는 스타일시트를 짠다면

가독성 측면에서 문제가 된다.

이걸 SCSS 코드로 바꿔보자.

$size: 70px !important;
$thickness: 6px !default;
$base-color: #d2d2d2 !default;
$wheel-color: #8d8d8d !default;
$speed: 800ms !default;

#loading-wrap {
    position: absolute;
    top: 0;
    left: 0;
    display: table;
    background-color: rgba(255, 255, 255, 0.69);
    width: 100%;
    height: 100%;
    border-radius: 10px;
    z-index: 999999;

    > div {
        display: table-cell;
        vertical-align: middle;

        #loading {
            height: $size;
            width: $size;
            border: $thickness solid $base-color;
            border: {
                right-color: $wheel-color;
                top-color: $wheel-color;
                radius: 100%;
            }
            margin: auto;
            animation: spin $speed infinite linear;
        }
    }
}

@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(359deg); }
}

각 선택자가 특정 선택자의 하위에 있다는 것을 직관적으로 알 수 있게 해준다.

최상단 라인에서 $ 기호를 이용해 변수를 지정하여 사용할 수 있으며,

CSS의 속성 또한 반복되는 부분에 대해 중첩하여

복잡한 스타일시트를 좀 더 편하게 작성할 수 있다.

& (상위 선택자 참조)

상위 선택자를 참조할 수 있는 문법으로,

& 문법을 이용해 hover나 active등의 가상클래스,

해당 선택자의 다른 클래스 선택자 등을 참조하여

중첩 구조를 좀 더 세분화할 수 있다.

input {
    &:[type=button] {
        font-size: 14px;
        background-color: blue;
        color: white;
        padding: 5px 10px;
        cursor: pointer;
        transition: all 150ms ease;
        
        &:hover,
        &.on {
            filter: brighness(1.2);
        }
    }
}

@if, @else if, @else

거의 SCSS는 하나의 프로그래밍 언어라 봐도 무방할 정도로 여러 문법을 지원한다. (신세계)

SCSS에서는 조건문을 통해 분기가 가능하다.

조건에 따라 스타일 속성을 달리 적용할 수 있다.

.app {
    @if (조건) {
        /* 스타일 속성 추가 */
    } @else if (조건) {
        /* 스타일 속성 추가 */
    } @else {
        /* 스타일 속성 추가 */
    }
}

@for, @each, @while

프로그래밍 언어처럼 조건문이 가능하므로 당연히 반복문도 가능하다.

/**
 * to일 경우 시작 ~ 종료 - 1
 * through일 경우 시작 ~ 종료
 * */
@for 변수 from 시작 [to | through] 종료 {
    /* 스타일 속성 추가 */
}

@each 변수 in 데이터 {
    /* 스타일 속성 추가 */
}

@while 조건 {
    /* 스타일 속성 추가 */
}
