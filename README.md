## Features

- 로그인 기능 (Firebase OAuth)
  - Endpoint: `/login`
  - Github, Google 로그인
  - 저 이외의 다른 사람은 로그인할 수 없습니다.
- 이력서 조회
  - Endpoint: `/resume`
- 이력서 수정 (저만 가능합니다)
  - Endpoint: `/admin/resume`
  - 회사 정보, 프로젝트 내용 등 추가/수정/삭제
