[React-Query] 사용법

2023-09-02

그 동안 Apollo만 오지게 썼었다가

React-Query 라는 좋은 도구가 나왔다는 이야기를 들었다.

써보니 편하고 괜찮더라고.

비슷한 형제로는 SWR, RTK Query 등이 있다.

장점

  • 캐시 관리가 (개인적으로는) Apollo보다 편했다.

단점

  • Apollo의 useLazyQuery가 React-Query에는 없다.

Setup

npm i react-query       # npm
yarn add react-query    # yarn

// src/pages/App.tsx

import { QueryClient, QueryClientProvider } from 'react-query';
import { ReactQueryDevtools } from 'react-query/devtools';

const client = new QueryClient();

export default function App() {
  return (
    <QueryClientProvider client={client}>
      <ReactQueryDevtools />
      {...}
    </QueryClientProvider>
  );
}

useQuery

C, R, U, D 중 R에 대한 api 요청

// src/pages/Foo/index.tsx

import { Suspense } from 'react';
import { useQuery } from 'react-query';

export default function Foo() {
  const { data, isLoading, refetch } = useQuery('foo', () => fetch('/api/foo'), {
    // 딴 데 갔다가 돌아왔을 때 refetch 시도 여부
    // 예) 크롬 탭에서 다른 탭 봤다가 다시 돌아왔을 때 refetch 할 것인지?
    refetchOnWindowFocus: false,
    // 요청 실패 후 재시도 횟수
    retry: 0,
    // 응답 성공
    onSuccess(data) {
      console.log('success:', data);
    },
    // 응답 실패
    onError(error) {
      console.log('error:', error);
    },
    // 응답 성공/실패 상관없이 실행
    onSettled(data, error) {
      console.log('success:', data);
      console.log('error:', error);
    },
  });
  
  return (
    <Suspense fallback={<div>...Loading</div>}>
      {...}
    </Suspense>
  );
}

Query Key

캐싱을 위한 고유 키 값

보통 useQuery의 첫 번째 인자로 들어가며, 문자열, 숫자로 넣을 수도 있고 리스트, 객체 형태와 같은 Object로도 가능하다.

이는 키 값으로 넣은 값에 대해 타입이 어떻든지 간에 useQuery 내부적으로 직렬화(Serialize)하기 때문이다.

useQuery('foo', () => fetch(...));
useQuery(['foo'], () => fetch(...));
useQuery(['foo', 1], () => fetch(...));
useQuery(['foo', { a: 1, b: 2 }], () => fetch(...));

useQueries

여러 쿼리를 동시에 실행

// src/pages/Foo/index.tsx

import { Suspense } from 'react';
import { useQueries } from 'react-query';

export default function Foo() {
  const [
    { data: a, isLoading: loadingA },
    { data: b, isLoading: loadingB },
    { data: c, isLoading: loadingC },
  ] = useQueries([
    {
      queryKey: 'a',
      queryFn() {
        return fetch('/api/a').then(response => response.json());
      },
    },
    {
      queryKey: 'b',
      queryFn() {
        return fetch('/api/b').then(response => response.json());
      },
    },
    {
      queryKey: 'c',
      queryFn() {
        return fetch('/api/c').then(response => response.json());
      },
    },
  ]);
  
  return (
    <Suspense fallback={<div>...Loading</div>}>
      {...}
    </Suspense>
  );
}
