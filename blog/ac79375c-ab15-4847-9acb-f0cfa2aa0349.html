Trouble Shooting

  1. Chunk 파일 로드 실패

    이런 에러가 떴었다.

    Unexpected token 'default'.

    이럴 땐 문제가 되는 코드의 export 부분을 다음과 같이 나누면 된다.

    // AS-IS
export default function Foo() { ... }

// TO-BE
function Foo() { ... }

export default Foo;

  2. OOM 이슈

    storybook을 빌드 또는 dev로 돌릴 때 OOM이 나면서 실행이 중단되었었는데,

    main.ts의 stories 프로퍼티 설정에서 glob 패턴을 되도록 좁혀야(?) 한다고 한다.

    나는 우선은 스토리 파일들을 주로 각 앱의 공통 컴포넌트에만 적용시킬꺼라

    경로를 components로 좁히기 위해 다음과 같이 해결하였다.

    import { StorybookConfig } from '@storybook/react-webpack5';

// AS-IS
const config: StorybookConfig = {
  stories: [
    '../../*/src/**/*.mdx',
    '../../*/src/**/*.stories.@(js|jsx|ts|tsx)',
  ],
  ...
};

// TO-BE
const config: StorybookConfig = {
  stories: [
    '../../native/src/components/**/*.mdx',
    '../../native/src/components/**/*.stories.@(js|jsx|ts|tsx)',
    '../../web/src/components/**/*.mdx',
    '../../web/src/components/**/*.stories.@(js|jsx|ts|tsx)',
  ],
  ...
};

export default config;
